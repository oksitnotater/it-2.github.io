Animasjonseksempel: trillebår som forflytter seg med rullende hjul

Elementer som legges inn: en ramme som forflyttes bortover (< div id = "ramme" >), to bilder (trillebår og ekstra hjul).
Generell stilsetting: justerer størrelser og avstander slik at hjulet legges oppå trillebårens hjul
Animasjonsstilsetting: hjulet ruller i animasjonen animation: roter 10s linear infinite. Rammen forflyttes og vendes i animasjonen animation: flytt 10s linear infinite.
@keyframes roter {
   0% { transform:rotate(0deg); }
   100% { transform:rotate(720deg); }
}
@keyframes flytt {
   0% { left: 0%;}
   45% {transform: rotateY(0deg);}
   50% {left: 70%; transform: rotateY(180deg);}
   95% {transform: rotateY(180deg);}
   100% { left: 0%; transform: rotateY(0deg);}
}
