Transfomasjoner med CSS

Transformasjoner er en måte å endre utseendet til et element uten at det påvirker resten av siden
I CSS gjøres dette med egenskapen transform, der egenskaper som forflytning (translate), rotasjon (rotate), skalering (scale) inngår
Dette kombineres gjerne med en gridstruktur for plassering av elementer.
Som vanlig: link til w3schools.com hvis dere vil ha mer info.

Oversikt over egenskaper og brukte verdier

Egenskap Beskrivelse Verdi/eksempel
translate Beskriver forflytning i xy-planettransform: translate{100px, 150px} flytter 100 px mot høyre og 150 px nedover
scaleBeskriver endring i størrelsetransform: scale(2, 0.5) dobler størrelse i x-retning og halverer i y-retning
rotate Beskriver en rotasjon av objektettransform: rotate(15deg) roterer 15 grader med klokka
transform: rotate(2turn) roterer 2 hele runder
skew Beskriver en forflytning av punktene transform: skew(20deg,10deg); forflytter punktene 20 og 10 grader
translateX, scaleY, rotateY, skewX,... Angir transformasjon i en retning
translate3d(x,y,z), scale3d(x,y,z), rotate3d(x,y,z), skew3d(x,y,z) Angir tredimensjonal transformasjon
initialTilbakestiller til standardverdier

Eksempel 1: bruk av transform

HTML-kode:
   < div id="flytt" >< /div >
   < div id="skaler" >< /div >
   < div id="roter" >< /div >
   < div id="punktforflytt" >< /div >
   < div id="samlet" >< /div >

CSS-kode:
   #flytt{
     background-color: pink;
     width: 400px;
     height: 200px;
     transform: translate(400px, 50px);
   }
   #skaler{
     background-color: lightgreen;
     width: 400px;
     height: 200px;
     transform: scale(0.4, 1.6);
   }
   #roter{
     background-color: purple;
     width: 400px;
     height: 200px;
     transform: rotate(-30deg);
   }
   #punktforflytt{
     background-color: orange;
     width: 400px;
     height: 200px;
     transform: skew(30deg);
   }
   #samlet{
     background-color: yellow;
     width: 400px;
     height: 200px;
     transform: translate(800px, -40px) scale(2, 0.8) rotate(20deg);
   }

Transformasjon med JavaScript - eksempel 2: boks som roterer når knapp trykkes

HTML-kode: Lager først boksen og en knapp for å rotere boksen:
   < div id="boks" > < /div >
   < button id="roterKnapp" > Roter < /button >

CSS-kode:
   #boks{
     width: 300px;
     height: 300px;
     background-color: lightBlue;
     border: 2px solid blue;
   }
Ved hjelp av en animasjon bestemmer jeg hvor ofte boksen skal rotere: jeg gir animasjonen 3 sekunder for en hel runde og deler en runde i 10 deler: gir ny vinkel hver 10% der vinkelen øker med 36 grader
   .rotereBoks{
     animation-name: rotasjon;
     animation-duration: 3s;
     animation-iteration-count: 1;
   }
Spesifiserer så at data for rotasjonsanimasjonen. Her bruker jeg transisjonen transform: rotate(n*36deg) som animasjonsargument.
   @keyframes rotasjon{
     10% {transform:rotate(36deg);}
     20% {transform:rotate(72deg);}
     30% {transform:rotate(108deg);}
     40% {transform:rotate(144deg);}
     50% {transform:rotate(180deg);}
     60% {transform:rotate(216deg);}
     70% {transform:rotate(252deg);}
     80% {transform:rotate(288deg);}
     90% {transform:rotate(324deg);}
     100% {transform:rotate(360deg);}
   }

JS-kode:
I script tillegger jeg boksen klassen rotereBoks når knappen trykkes:
   var knappEl = document.querySelector("#roterKnapp");
   var boksEl = document.querySelector("#boks");
   knappEl.addEventListener("click", roterFunksjon);
   function roterFunksjon(){
     boksEl.classList.add("rotereBoks");
   }
