Diagrammer med canvas

Dette krever litt kunnskaper om array og objects. Eksamensrelevant tema: de to bruksområdene for canvas er først og fremst animasjon og diagrammer

Først må vi kjenne til kommandoen for å skrive tekst i et canvas:
   ctx.font ="24 px 'Arial' "; bestemmer skrift (merk ' ')
   ctx.fillStyle ="blue"; bestemmer skriftfarge
   ctx.fillText("teksten jeg skriver", xPos, yPos); setter inn tekst

   var canvas5=document.querySelector("#canvas5");
   var ctx5=canvas5.getContext("2d");
   ctx5.font ="30px 'Arial' ";
   ctx5.fillStyle = "blue";
   ctx5.fillText("teksten jeg skriver", 50, 50);

Til å lage et søylediagram trenger vi et datasett å ta utgangspunkt, for eksempel antall timer skolearbeid hver dag. Dette lagres som en array med objekter, der egenskapene er dag og timer:
   var data = [
     {dag: "Mandag", timer: 8},
     {dag: "Tirsdag", timer: 7},
     {dag: "Onsdag", timer: 8},
     {dag: "Torsdag", timer: 6},
     {dag: "Fredag", timer: 6},
     {dag: "Lørdag", timer: 4},
     {dag: "Søndag", timer: 1}
  ];

I tillegg lagrer vi spesifikasjoner for tekst og søyler som variabler:
   var tekstBredde = 250;
   var totalBredde = canvas5.width; (samme bredde som canvaset)
   var soyleBredde = totalBredde - tekstbredde - 10; (bestemmer avgrensning for søylene)
   canvas5.height = data.length*30; (endrer størrelsen på canvaset avhengig av antall verdier i arrayen)
   var max = 0; (denne kommer vi til å endre til verdien til den største søylen, noe vi gjør med en for-løkke:)
   for(var i=0; i < data.length; i++){
     if (data[i].timer > max) {
       max = data[i].timer;
    }
  }

Videre spesifiseres utseendet:
   ctx5.font ="30px 'Arial' ";
   ctx5.textAlign = "end"; (høyrejustert tekst)
   ctx5.textBaseline = "hanging"; (plasserer teksten under punktet som angis som plassering)

Nå kan vi tegne søyler som representerer datasettet:
   for (var i = 0; i < data.length; i++){
     //1: Bestemmer skriftfarge så alle dager får samme
     ctx5.fillStyle = "#323232";
     //2: Lager en variabel som bestemmer størrelsen på den aktuelle søylen:
     var storrelse = (data[i].timer/max)*soyleBredde;
     //3: Skriver ut hvilken dag og hvor mange timer
     ctx5.fillText(data[i].dag + '(' + data[i].timer + ')', tekstBredde, (30*i) + 3); (husk: tekst, xPos, yPos)
     //4: Sørger for at hver søyle får ny farge, bruker hsl-verdi
     ctx5.fillStyle = 'hsl('+i*30 +' , 100%, 70%)';
     //5: Tegner søylene
     ctx5.fillRect(tekstBredde + 5, (30*i)+3, storrelse, 24);
   }

Oppgave:

Hvordan kan dere bruke dette sammen med kunnskaper om tegning av sirkeler til å tegne sektordiagram?
1. Finn totalt antall timer
2. Finn andelen hver dag skal ha av sirkelen, og sammenlikne dette med andel av 2PI
3. Husk at hver sektor må ha startpunkt der den forrige har sluttpunkt

Løsning

   var canvas7=document.querySelector("#canvas7");
   var ctx7=canvas7.getContext("2d");
   // Lagrer totalt antall timer i en variabel:
   var antTimer = 0;
   for(var i=0; i < data.length; i++){
     antTimer += data[i].timer;
   }
   // Finner andel av total for hver dag i en array:
   var andel = [];
   andel.length = data.length;
   for(var i = 0; i < data.length; i++){
     andel[i] = data[i].timer/antTimer;
   }
   console.log(andel);
   // Gjør dette om til total vinkel for dagen:
   var vinkel = [];
   vinkel.length = andel.length;
   var sum = 0;
   for(var i = 0; i < data.length; i++){
     vinkel[i] = andel[i]*2*Math.PI;
   }
   console.log(vinkel);
   // vi kunne gjort dette enda enklere ved å legge til nye egenskaper (andel og vinkel) i den opprinnelige objecten!
   // plasserer sektordiagrammet litt på siden
   var cX = canvas7.width/3;
   var cY = canvas7.height/2;
   var radius = 150;
   // Lager en variabel som lagrer hvor forrige sektor sluttet for å starte neste riktig sted
   var forrigeSlutt = 0;
   // Går gjennom alle verdiene og tegner kakestykker for hver av dem
   for (var i = 0; i < data.length; i++) {
     var sektorStr = vinkel[i];
     var sektorSlutt = forrigeSlutt + sektorStr;
     // Tegner sektoren
     ctx7.beginPath();
     ctx7.arc(cX, cY, radius, forrigeSlutt, sektorSlutt);
     ctx7.lineTo(cX, cY);
     ctx7.fillStyle = "hsl("+i*30 +" , 100%, 50%)";
     ctx7.fill();
     ctx7.closePath();
     // Oppdaterer variabelen forrigeSlutt
     forrigeSlutt = sektorSlutt;
     // Angir tekststørrelse og type for teksten og plasserer den ved siden av sektordiagrammet
     ctx7.font = "24px 'Century Gothic'";
     ctx7.textBaseline = "hanging";
     var tekstStartX = canvas7.width / 1.7;
     var tekstStartY = (canvas7.height / 2) - (30 * (data.length / 2)) + (30 * i);
     ctx7.fillText(data[i].dag + "(" + data[i].timer + ")", tekstStartX, tekstStartY);
   }
