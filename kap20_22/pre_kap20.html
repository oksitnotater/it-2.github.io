Set og map

Arrays og objects er to av mage typer datakolleksjoner. Set og map er to nye vi skal bli kjent med i dag

Arrays

Arrays gir indekserte verdier, der vi han hente ut hver enkel verdi ved å referere til plasseringen (nummeret) til den aktuelle verdien. Vi husker at nummereringen starter på 0. Vi kan enten legge inn verdier når vi oppretter arrayen eller opprette en tom array som siden fylles med verdier.
Vi husker hvordan vi selv sorterte arrays ved å implementere metoden bubblesort, men legg også merke til at vi har følgende metoder for å sortere:

Metode Forklaring
arrayNavn.sort(); Sorterer tekst alfabetisk
arrayNavn.sort(function(a,b){return a-b}); Sorterer tall i stigende rekkefølge
arrayNavn.sort(function(a,b){return b-a}); Sorterer tall i synkende rekkefølge
arrayNavn.reverse() Bytter rekkefølge

Dersom vi skal legge inn nye verdier i arrayen har vi ulike metoder, under beskrives disse.

Metode Forklaring
arrayNavn.push(verdi) Legg til en ny verdi på slutten av arrayen
arrayNavn.unshift(verdi) Legg til en ny verdi på begynnelsen av arrayen
arrayNavn.pop() Ta bort arrayens siste verdi
arrayNavn.shift() Ta bort arrayens første verdi
arrayNavn.splice(indeks, antall som fjernes) Ta bort verdier i en gitt indeks
arrayNavn.splice(indeks, antall som fjernes,verdi) Ta bort verdier i en gitt indeks og sett inn en ny verdi

Set

En set-kolleksjon er en kolleksjon av data som har unike verdier: ingen verdier er like. Vi kan altså lage array i alle tilfeller vi kan lage set, men ikke omvendt. I og med at ingen set-verdier er like, er det lett å sjekke om seten inneholder bestemte verdier, hente ut verdier og slette bestemte verdier.

Metode Forklaring
var setNavn = new Set() Oppretter ny set
setNavn.add(verdi) Legger til en verdi
setNavn.clear() Sletter alle verdier
setNavn.delete(verdi) Sletter en bestemt verdi
setNavn.has(verdi) Sjekker om en bestemt verdi finnes
setNavn.values() Gir ut en liste med alle verdiene
setNavn.size Gir ut antall verdier i kolleksjonen

Eksempel: vi oppretter en set-kolleksjon som inneholder ulike stjerner
   var stjerner = new Set(); // oppretter set
   stjerner.add("Sirius"); // legger inn verdier
   stjerner.add("Vega");
   stjerner.add("Pollux");
   document.write(stjerner.has("Arcturus")); // sjekker om stjernen Arcturus finnes
   document.write(stjerner.size); // skriver ut størrelsen
   stjerner.add("Pollux"); // forsøker å legge til Pollux en gang til
   document.write(stjerner.size);
   stjerner.delete("Pollux"); // sletter Pollux
   document.write(stjerner.size);
   stjerner.add("Pollux"); // forsøker å legge til Pollux igjen
   document.write(stjerner.size);
   stjerner.clear(); // sletter alt innhold
   document.write(stjerner.size); //

Set og array

Det går an å opprette en set med verdiene fra en array:
   var arraySet = new Set([verdi1, verdi2, verdi3]);
Set-metodene fungerer akkurat som vanlig. Et eksempel vises under:
   var filmer = new Set(["Harry Potter", "Hobbiten", "Fabeldyr"]);
   if(filmer.has("Hobbiten")){
   document.write("Hobbiten finnes i samlingen");
   filmer.add("Ringenes Herre");
   document.write("Alle som eier Hobbiten eier også Ringenes Herre. Denne er lagt til");
   }

Set og array har ulike fordeler: set gir enklest muligheten til å hente ut verdier mens array er mulig å sortere. Det er derfor aktuelt å konvertere mellom array og set:

Metode Forklaring
var setNavn = new Set(arrayNavn) Oppretter ny set fra array
var arrayNavn = Array.from(setNavn) Oppretter ny array fra set

Transversere en set

Å transversere en set vil si å jobbe seg gjennom en set verdi-for-verdi: dersom vi ønsker å utføre den samme operasjonen på alle elementene (f.eks. skrive ut alle verdiene i en tekst). Her bruker vi en for-of-løkke:
   for (var elementNavn of setNavn){
   kode som skjer
  }
Et eksempel er hvis vi har en set med ulike blomster og vil liste dem opp:
   var blomster = new Set(["Solsikke", "Engsoleie", "Løvetann", "Hestehov"]);
   document.write("< ul >")
   for(var blomst of blomster){
   document.write("< li >" + blomst + "< /li >");
  }
   document.write("< /ul >")

Map

Vi så at vi kan bruke set i stedet for arrays der det kun er unike verdier og der rekkefølgen ikke spiller noen rolle
Nå skal vi gå gjennom datakolleksjonen map, som kan minne om objects. I map hører alle verdier til en nøkkel, der vi bruker nøkkelen til å få tak i verdien. Vi har flere av de samme metodene i map som i set.

Metode Forklaring
var mapNavn = new Map() Oppretter ny map
mapNavn.set(nøkkel, verdi) Legger til en verdi
mapNavn.clear() Sletter alle verdier
mapNavn.delete(nøkkel) Sletter en verdi med en bestemt nøkkel
mapNavn.has(nøkkel) Sjekker om en verdi med en bestemt nøkkel finnes
mapNavn.size Gir ut antall verdier i kolleksjonen
mapNavn.values() Gir ut en liste med alle verdiene
mapNavn.keys() Gir ut en liste over alle nøklene
mapNavn.entries() Gir ut en liste med alle nøkkel- og verdipar
mapNavn.get(nøkkel) Henter ut en verdi med en bestemt nøkkel

Eksempel: vi registrerer nye personer i en map.
Her vil nøkkelen være personnummeret: denne er unik for hver innbygger
   //oppretter map
   var innbyggere = new Map();
   innbyggere.set("001", "Kalle Kanin");
   innbyggere.set("002", "Tassen");
   innbyggere.set("003", "Klaus Ku");
   //henter ut antall innbyggere:
   document.write(innbyggere.size);
   //sjekker om vi har en innbygger 002 og 005
   document.write(innbyggere.has("002") + "< br >" + innbyggere.has("005"));
   //sletter Kalle Kanin
   innbyggere.delete("001");
   //erstatter Klaus Ku med Klara Ku
   innbyggere.set("003", "Klara Ku");
   //skriver ut Klara Ku
   document.write(innbyggere.get("003"));

Map og objects

Vi kan også opprette en map fra objects med informasjon. Som et eksempel har vi en variabel for Kalle Kanin, Tassen og Klaus Ku:
   // oppretter objects med informasjonen
   var kalleKanin = {
     pnr: "001",
     navn: "Kalle Kanin",
     adresse: "Kaninsvingen 1"
   }
   var tassen = {
     pnr: "002",
     navn: "Tassen",
     adresse: "Kjøttbeinstien 1"
   }
   var klausKu = {
     pnr: "003",
     navn: "Klaus Ku",
     adresse: "Kvegfaret 1"
   }
  // oppretter en map der vi skal legge inn informasjonen
   var innbyggere = new Map();
   innbyggere.set(kalleKanin.pnr, kalleKanin);
   innbyggere.set(tassen.pnr, tassen);
   innbyggere.set(klausKu.pnr, klausKu);

Transversere en map

Vi har tre muligheter til å jobbe oss gjennom en map: enten ved å bruke nøkler, verdier eller begge. De ulike metodene vises under:

Metode Forklaring
for(var nøkkel of mapNavn.keys()){kode} Transverserer nøklene
for(var verdi of mapNavn.values()){kode} Transverserer verdiene
for (var [verdi, nøkkel] of mapNavn){kode} Transverserer både nøkler og verdier
mapNavn.forEach(function(verdi, nøkkel){kode}) Transverserer både nøkler og verdier

Eksempel: Lager en map med filmer (nøkkel er IMDB sin) og løkker gjennom den på tre måter:
   var filmer = new Map();
   filmer.set("tt0241527", "Harry Potter og de vises stein");
   filmer.set("tt0120737", "Ringens brorskap");
   filmer.set("tt3183660", "Fabeldyr og hvor de er å finne");
   // løkker gjennom nøklene
   for(var filmId of filmer.keys()){
     document.write(filmId + "< br>");
   }
  // løkker gjennom verdiene
   for (var film of filmer.values()){
     document.write(film + "< br>");
   }
   // løkker gjennom begge
   for(var [filmId, film] of filmer){
     document.write(filmId + ":" + film + "< br>");
   }
   filmer.forEach(function(film, filmID){
     document.write(filmId + ":" + film + "< br>");
   })

forEach

Kan også brukes for arrays og sets:

Kolleksjon forEach-syntax
Array arrayNavn.forEach(function(verdi, indeks, array){kode})
Set mapNavn.forEach(function(verdi, verdi, set){kode})
Map mapNavn.forEach(function(verdi, nøkkel, map){kode})
