Introduksjon til objektorientert programmering i JavaScript

Forskjellen på prosedyreorientert og objektorientert programmering

Så langt i faget har vi holdt oss til prosedyreorientert programmering: vi har skrevet kode med metoder som følger hverandre. Vi har likevel hilst litt på det objektorienterte ved å ta i bruk objekter, men foreløpig kun for å organisere informasjon med tilhørende nøkler.
Vi vil de neste timene ta i bruk objekter og klasser for å organisere kode i større grad, blant annet for å enklere gjenta kode.

Objektorientert programmering handler om å organisere kode i klasser og objekter. En klasse kan sammenliknes med en arkitekttegning av et hus, mens objektene kan sammenliknes med husene som er byggget etter tegingen: klassen gir "oppskriften" og objektene er produkter av oppskriften. Vi kan si dette som at objektene er konkrete tilfeller av klassen.

Implementere klasser i koden

En klasse har kodeordet class, et klassenavn (stor forbokstav), kode som står mellom {}. I koden har vi verdier, kalt attributter, og funksjonalitet, kalt metoder. Koden må ha en konstruktør som definerer hvilke attributter som tilhører klassen.
Syntaxen for å definere en klasse er dermed:

   class Klassenavn{
     constructor(parameter1, parameter2){
     this.attributt1 = parameter1;
     this.attributtt2 = parameter2;
    }
  }

Som et eksempel kan vi lage en klasse for forfattere: parametrene blir her fornavn, etternavn og fødeslsår. Videre skal vi se hvordan vi kan legge til objekter (bestemte forfattere) til denne klassen.

   class Forfatter{ // oppretter klasse
     constructor(fnavn, enavn, faar){ // attributtene til klassen
       this.fnavn = fnavn; // attributtet fornavn får paramenterverdien fornavn
       this.enavn = enavn;
       this.faar = faar;
    }
  }

Legge til objekter i klassen

Objektene i forfatterklassen vil være forfattere. Syntaxen for å legge et objekt til en klasse er
   let objektNavn = new Klassenavn(parameter1, parameter2);
For en forfatter vil det si
   let blixen = new Forfatter("Karen", "Blixen", 1885);
   let hamsum = new Forfatter("Knut", "Hamsun", 1859);

Vi kan skrive ut informasjon om forfattene med
   document.write(blixen.fnavn + " " + blixen.enavn + " er født i " + blixen.faar);

Legge til metoder i klassen

I stedet for å skrive samme kode for å skrive ut hver enkelt forfatter, kan vi lage en metode som skriver ut forfatterinformasjon. Metoden skrives inn i klassen som
   metodeNavn(){
     //kode som skal skrives ut
  }
For forfatterne blir klassen da
   class Forfatter{ // oppretter klasse
     constructor(fnavn, enavn, faar){ // attributtene til klassen
       this.fnavn = fnavn; // attributtet fornavn får paramenterverdien fornavn
       this.enavn = enavn;
       this.faar = faar;
    }
     visInfo(){// metoden kaller jeg visInfo
       document.write(this.fnavn + " " + this.enavn + " er født i " + this.faar); // this peker på hvilken forfatter
    }
  }

Vi kaller på metoden ved å skrive
   blixen.visInfo();
