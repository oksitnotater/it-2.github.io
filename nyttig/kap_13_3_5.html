Video og lyd

Video

Taggen for å legge til video er < video > < /video>, mens selve filmen legges inn i < source >: på samme måte som < figure > og < img > for bilder:
   < video >
     < source src = "filnavn.filendelse" type = "video/mp4">
   < /video >

I tillegg til src og type har vi andre attributter det kan være aktuelt å inkludere:
   autoplay - sørger for at filmen spilles automatisk når siden lates
   controls - legger inn play/pause/lyd
   width eller height - styrer størrelsen
   loop - gjentar avspilling
   muted - starter uten lyd

Så vi kan f.eks. legge ut en film som spiller automatisk, ikke har kontroller og looper:
   < video width="1200px" autoplay loop muted>
     < source src="media/flikkflakk.mp4" type="video/mp4" >
     Din nettleser støtter ikke videoelementet
   < /video >

Legge til egne filmer

Når vi legger til filmer som ligger på egen maskin, må vi passe på filadressen som vanlig. Av hensyn til den som bruker siden, legger vi på controls. Dersom vi ikke spesifiserer annet, legges filmene inline (ved siden av hverandre).


Legge til youtube-filmer

I html har vi mulighet til å legge til youtubefilmer direkte i stedet for kun å legge ut en link til filmen:

Resultatet blir da
   < iframe width="560" height="315" src="https://www.youtube.com/embed/n1ZOv90AytQ" frameborder="0" allowfullscreen >< /iframe >

Lyd

Taggen for å legge til lyd er < audio > og brukes, som film, sammen med < source > for å sette inn lydfiler:
   < audio controls >
     < source src="media/Beatles_Twist_and_Shout.mp3" type = "audio/mpeg">
     Din nettleser støtter ikke audioelementet
   < /audio >

Å bruke JavaScript til å styre lyd og video

Vi kan hente ut lyd- og videoelementer på samme måte som alle andre elementer: ved å referere til type element, klasse eller id. Deretter kan vi styre egenskaper som play, pause, autoplay og loop med javascript ved f.eks. knapper. Disse knappene kan erstatte eller komme i tillegg til controls (husk at controls også gir mulighet til å spole, justere lyd osv)
   < video width="500px" muted id = "flikkflakk" >
     < source src="media/flikkflakk.mp4" type="video/mp4">
     Din nettleser støtter ikke videoelementet
   < /video >< br >

   < button id="spill" >Spill film< /button >
   < button id="gjenta" >Loop film< /button >
   < button id="stopp" >Stopp film< /button >

   < script >
     var flikkflakkFilm = document.querySelector("#flikkflakk");
     var spillKnapp = document.querySelector("#spill");
     var loopKnapp = document.querySelector("#gjenta");
     var stoppKnapp = document.querySelector("#stopp");
     spillKnapp.addEventListener("click", spillFilm);
     function spillFilm(){
       flikkflakkFilm.play();
     }
     loopKnapp.addEventListener("click", gjentaFilm);
     function gjentaFilm(){
       flikkflakkFilm.loop=true;
     }
     stoppKnapp.addEventListener("click", stoppFilm);
     function stoppFilm(){
       flikkflakkFilm.pause();
     }
  < /script >
