Å legge videoer i forgrunnsvindu

Vi kan ønske å legge videoer i forgrunnsvindu for eksempel hvis det skal være en reklamefilm som må vises før vi får gå inn på selve nettstedet eller dersom vi ikke ønsker at en film skal fylle opp en nettside før brukeren faktisk skal se den. Egenskapene vi bruker for å sette videoer i forgrunnsvindu er de samme som når vi lager dropdownmenyer (for de av dere som har gjort det): vi justerer z-index og display.
I dette eksempelet skal vi vise det siste eksempelet: en film som åpnes i et forgrunnsbilde når brukeren velger å se den.

Vi legger først til en knapp brukeren kan trykke på for å få opp filmen:
   < button id ="visFilm" >Vis film< /button >

Selve filmen legges til i et div-element som vi justerer css-egenskapene til. Her finnes også en lukk-knapp:
   < div id = "videoDiv" >
     < video width="576px" height="320px" controls >
       < source src="media/turn_2.mp4" type="video/mp4" >
     < /video >
     < button id="lukkFilm" > X < /button >
   < /div >

I css justeres egenskapene til forgrunnsvinduet. Her er dispay: none (filmen synes i utgangspunktet ikke), position: absolute og z-index: 1000 (filmen legges foran alt annet innhold når display endres):
   #videoDiv{
     background-color: #DDDDDD;
     padding: 10px;
     border: 2px solid black;
     display: none;
     position: absolute;
     z-index: 1000;
     left: 0;
     top: 0;
     width: 576px;
     height: 320px;
   }
   #lukkFilm{
     margin-top: 15px;
   }

Til vis-knappen legges en lytter som endrer display til "initial" og plasserer filmen på siden ved bruk av window.innerHeight og window.innerWidth, og til lukk-knappen legges en lytter som endrer display tilbake:
   < script >
     var visFilm = document.querySelector("#visFilm");
     var videoDiv = document.querySelector("#videoDiv");
     var lukkFilm = document.querySelector("#lukkFilm");
     visFilm.addEventListener("click", visFilmFunk);
     lukkFilm.addEventListener("click", lukkFilmFunk);
     function visFilmFunk(){
       videoDiv.style.display = "initial";
       var topp = (window.innerHeight - 320)/2;
       var venstre = (window.innerWidth - 576)/2;
       videoDiv.style.top = topp+"px";
       videoDiv.style.left = venstre+"px";
     }
     function lukkFilmFunk(){
       videoDiv.style.display = "none";
     }
   < /script >

Oppgaver

1. Lag en nettside der videovinduet vises automatisk med en lukk-knapp for å komme videre til selve nettsiden. Videoen skal dekke 90% av bredden til nettleservinduet. Det skal gå an å åpne opp igjen videoen.
2. Utvid koden slik at filmen settes på pause når forgrunnsvinduet lukkes og fortsetter der den slapp hvis brukeren åpner videoen igjen.
