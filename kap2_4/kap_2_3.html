Aritmetiske operasjoner på string

Dersom vi foretar en aritmetisk operasjon på tall lagret som en tekststreng, vil følgende skje:
   < script >
     var tekstVar = 'Tallet er ';
     var tallVar1 = '42';
     var tallVar2 = '17';
     document.write(tekstVar + tallVar1 + tallVar2);
   < /script >
Aritmetiske operasjoner har med andre ord ikke mye for seg på tall lagret i tekststrenger. Skal vi bruke aritmetiske operasjoner, må tall lagres som number

Oppgave

Skriv inn følgende kode og legg merke til hva som skrives til konsollen
   < script >
     // 1
     var var1 = 'Tallet er ';
     var var2 = 42;
     var var3 = 17;
     document.write(var1 + var2 + var3 +'< br >');
     //2
     document.write(var1 + (var2+var3) +'< br >')
     //3
     var1 = 42;
     var2 = 17;
     var3 = 'er tallet';
     document.write (var1 + var2 + var3 +'< br >');
   < /script >

Hvordan vet vi hva som er datatypen?

Vi kan bruke metoden typeof til å finne ut hva slags datatype en variabel har med metoden typeof:
   < script >
     var tekstVar = "Her skriver jeg tekst";
     console.log(typeof tekstVar); // returnerer string
     var tallVar = 42;
     console.log(typeof tallVar); // returnerer number
     var tallTekstVar = '42';
     console.log(typeof tallTekstVar); // returnerer string
     var boolskVar = false;
     console.log(typeof boolskVar); // returnerer boolean
   < /script >

Endre datatype

Dersom vi oppdager at en variabel er lagret som feil datatype, kan vi gjøre om mellom de to. Dette gir oss f.eks. mulighet til å sjekke antall siffer i et tall. Husk at det ikke gir mening å gjøre om tekst til number.
var nyVariabel = Number(gammelVariabel) gjør om til tall
var nyVariabel = String(gammelVariabel) gjør om til tekst
   < script >
     var tekstTallVar = '42';
     var tallTekstVar = 3489;
     console.log(tekstTallVar + typeof tekstTallVar);
     console.log(tallTekstVar + typeof tallTekstVar);
     var tallFraTekst = Number(tekstTallVar);
     var tekstFraTall = String(tallTekstVar);
     console.log(tallFraTekst + typeof tallFraTekst);
     console.log(tekstFraTall + typeof tekstFraTall + tekstFraTall.length);
   < /script >

Aritmetiske operasjoner på number

De aritmetiske operasjonene inneholder blant annet de fire regneartene, i tillegg har vi tre tilleggsoperasjoner:

Operator Forklaring Eksempel
+ Addisjon add = a + b;
- Subtraksjon sub = c - b;
* Multiplikasjon multi = b * c;
/ Divisjon divi = c / b;
% Modulus (restregning) rest = c % a;
++ Inkrement inkr = a++; (dvs inkr = a + 1)
-- Dekrement dekr = b--; (dvs dekr = b - 1)

Modulo-operatoren

Modulo handler om restregning: her blir 12 % 10 = 2 fordi vi har 2 i rest ved 12 / 10. Dette kan f.eks brukes til å finne
   Partall: alle tall der tall % 2 = 0
   Oddetall: alle tall der tall % 2 = 1
   Tall delelig med 7: alle tall der tall % 7 = 0
Osv.

Inkrement og dekrement

a++ og b-- forenkler skrivemåten der vi skal øke eller minke variabler med én. Dette er spesielt nyttig i løkker, der vi øker eller minsker telleren for hver gang løkken kjøres.

Tilordningoperatorer

Å si at
   var a = 2;
   var a = a + 3;
Vil si at vi tilordner verdien a tallet 5 (i stedet for 2). Slike regnestykker kan skrives enklere:

Operator Eksempel Tilsvarer
= a = 1; a = 1
+= a += 5; a = a + 5 = 1 + 5 = 6
-= a-= 1; a = a - 1 = 6 - 1 = 5
*= a*= 4; a = a * 4 = 5 * 4 = 20
/= a /= 2; a = a / 2 = 20 / 2 = 10
%= a %= 3; a = a % 3 = 10 % 3 = 1

Tilordningsoperatoren += er også nyttig for tekst, ved at vi kan legge til nye ledd i tekststrengen fortløpende:

   < script >
     var tekst = 'Tekststrengen starter med litt tekst, ';
     tekst += 'før vi legger til mer tekst ';
     tekst += 'og slik kan vi fortsette.';
     document.write(tekst);
   < /script >
