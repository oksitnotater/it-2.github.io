Kapittel 1

Hva er JavaScript?

JavaScript er et programmeringsspråk som nettleseren oversetter til maskinkode. Mens HTML styrer innhold på en nettside og CSS styrer utseende til en nettside, styrer JS interaktiviteten til en nettside.
Hva kan vi gjøre med JavaScript?

  • Endre innhold i HTML-elementer
  • Lage nye HTML-elementer
  • Ta bort HTML-elementer
  • Endre CSS-stiler
  • Gjøre beregninger
  • Hente inn data fra andre datakilder
  • Legge inn kart på HTML-sider
  • Spille av lyd ved hendelser (klikke på siden, scrolle osv)
  • Tegne/animere vektorgrafikk
  • Lage animasjoner
  • Og mye mer

Hvordan bruker vi JavaScript?

Vi kan enten lage egne JS-dokumenter som vi linker til i hoveddokumentet (external), eller legge kode direkte i HTML-fila (internal). Vi starter med internal-metoden. All javascript legges da mellom taggene < script > og < /script >:

Vi starter med fire måter å presentere informasjon på:
   < script >
     document.write('dette vil synes på siden');
     console.log('dette vil synes i konsollen, som vi finner ved ctrl+shift+j');
     alert('oj, det virker!');
     prompt('nå stiller jeg et spørsmål til deg');
   < /script >

Kommentarer i JavaScript

Vi husker kommentarer i html: < !-- dette leses ikke som kode -- >
Og css: /* dette leses ikke som kode */

I js har vi to muligheter: // korte kommentarer
og /* lengre kommentarer */

console.log og document.write

Metoden document.write('tekst'); ser vi at skriver til skjermen. Dette er en fin metode dersom vi ønsker å skrive til skjerm, men fungerer dårligere dersom vi for eksempel skal kontrollere en mellomregning underveis i et større script, eller sjekke om kode kjører. Da bruker vi heller konsollen:
Dersom dere klikker ctrl+shift+J i chrome, kommer dere inn på konsollen. Den er i utgangspunktet tom, frem til du logger dit. Vi logger til konsollen med å skrive
   < script >
     console.log('Dette er en test');
   < /script >

Oppgaver

Opprett et nytt HTML-dokument. Skriv inn følgende kode og legg merke til hva som skjer:
   < script >
     document.write('< h1 > Dette skrives som en overskrift < /h1 >' +
     '< p> Jammen fungerer ikke html-tagger også i document.write < /p >');
     console.log('men i konsollen kan jeg bare skrive tekst uten stilsetting');
   < /script >

Prøv så å sette inn andre html-elementer ved hjelp av document.write.
