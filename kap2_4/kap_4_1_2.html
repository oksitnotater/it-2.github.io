Hva bruker vi funksjoner til?

JavaScript inneholder mange forhåndsdefinerte funksjoner som vi tar i bruk: for å lage tilfeldige tall, for å skrive til skjerm, for å skrive til konsoll også videre. Disse funksjonene kaller "egne programmer" som utfører en viss operasjon. I tillegg har vi muligheter til å skrive våre egne funksjoner som utfører en gitt kode. Dette kan være kode som kjører når en nettside åpnes, kode som kjører når en knapp trykkes, eller andre.
Når vi deklarerer en funksjon, må vi gi den et funskjonsnavn. Funksjonsnavnet skal beskrive funksjonen, og må ikke være likt noen innebygde funksjoner. Det finnes to måter å deklarere funksjoner på: enten direkte, eller ved å lagre dem til en variabel:
   < script >
     function funksjonsNavn(){
       // kode som kjøres
     }
   < /script >

   < script >
     var funksjonsNavn = function(){
       // kode som kjøres
     }
   < /script >

Jeg kommer til å bruke den første måten i undervisningen.
Når vi skal kalle en funksjon, altså bruke den, bruker vi funksjonsnavnet:
   < script >
     funksjonsNavn();
   < /script >

Eksempel: en funksjon som skriver ut gangetabellen

   < script >
     function gangetabell(){
       document.write('< table >');
       for (var i=1; i<=10; i++){
         document.write('< tr >');
         for(var j=1; j<=10; j++){
           document.write('< td >'+i*j+'< /td >');
         }
         document.write('< /tr >');
       }
       document.write('< /table >');
     }
   gangetabell();
   < /script >

Funksjoner med argumenter

Når vi bruker funksjoner kan vi også gi dem et argument, som vil bestemme resultatet av koden. Vi kan for eksempel lage en funksjon som beregner areal og omkrets av en sirkel, der radiusen er argument.

I denne funksjonen brukes metoden toFixed(2) for å gi svaret med to desimaler

   < script >
     function sirkel(r){
       areal = (Math.PI*r*r).toFixed(2);
       omkrets = (2*Math.PI*r).toFixed(2);
       console.log('Arealet til en sirkel med radius '+r+' er '+areal+' og omkretsen er '+omkrets+'.');
     }
     sirkel(4);
     sirkel(1);
     sirkel(10);
   < /script >

Oppgave

Utvid funksjonen gangetabell til at den tar inn størrelsen på gangetabellen (i stedet for standard 10*10) og returnerer en tabell med denne gangetabellen.
Side 60: 4.1, 4.2, 4.3
Side 62: oppgave 4.6, 4.7
