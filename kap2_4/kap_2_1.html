Variabler

Variabler brukes til å holde orden på verdier underveis i et program. Slike verdier kan være tekst (f.eks. navn), tall (f.eks. konstanter) eller de boolske verdiene sann eller usann.
For å fortelle programmet at vi lagrer en variabel, bruker vi nøkkelordet var:
   < script >
     var fornavn = 'Asta';
     var etternavn = 'Fjærli';
     var alder = 28;
     // disse variablene kan vi bruke i f.eks. document.write() og console.log();
     document.write('navnet er fornavn etternavn og alderen er alder år');
     document.write('navnet er ' + fornavn + ' ' + etternavn + ' og alderen er ' + alder + ' år');
   < /script >

Oppgave

Skriv inn koden over i dokumentet ditt fra i sta, men bytt ut med ditt eget navn og alder. Hva betyr + ' ' + (evt + " " +) mellom fornavn og etternavn?

Overskriving av variabler

Når vi først har lagret en variabel med nøkkelordet var, kan vi overskrive den uten å bruke nøkkelordet var:
   < script >
     var tall = 0;
     tall = 42;
     console.log(tall);
   < /script >

Oppgave

Skriv inn koden over og se hva som skrives til konsollen.

Variabelnavn

Regler for å sette variabelnavn:

  • Variabelnavn er case sensitive: var TALL og var tall er to forskjellige variabler
  • Navnet kan ikke starte med et tall
  • Navnet kan ikke starte med stor forbokstav
  • Variabelnavn kan ikke hete det samme som metoder i JS: f.eks. ikke hete var console = 'konsollen'
  • Dersom variabelnavnet består av flere ord, gir vi det enten storForbokstav for neste ord, eller deler_med_understrek
  • Variabelnavn skal ikke inneholde punktum, komma, bindestrek eller andre tegn
  • Å bruke æøå i variabelnavn er lite ønskelig, da det begrenser bruk av koden med utenlandsk tastatur
  • Variabelnavn av typen matematiske konstanter lagres med kun store bokstaver: f.eks. Math.PI
