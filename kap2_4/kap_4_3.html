Funksjoner med returverdier

Så langt har funksjonene våre gitt oss et "svar" i form av å skrive til konsollen eller direkte til nettsiden. Vi kan også ha funksjoner som fungerer mer som en mellomregning: som gir oss en returverdi vi kan lagre inn i en variabel og bruke videre.

Som et eksempel kan vi bruke funksjonen som lager et tilfeldig tall mellom en nedre og øvre grense:
   < script >
     function tilfeldigTall(min, max){
       document.write(Math.floor(Math.random()*(max - min + 1)+min));
     }
     tilfeldigTall(2, 10);
   < /script >


Det tilfeldige tallet som blir generert her kan vi la være returverdien til funksjonen. Dette kan gjøres ved å legge inn en return i funksjonen, og så lagre svaret i en variabel:
   < script >
     function tilfeldigTall(min, max){
       return(Math.floor(Math.random()*(max - min + 1)+min));
     }
     var tilfeldig = tilfeldigTall(2, 10);
     document.write(tilfeldig);
   < /script >

Return avslutter funksjonen

En annen egenskap med å bruke return, er at funksjonen automatisk avsluttes her. All kode i funksjonen som ligger etter return, vil ikke bli lest. Dette kan brukes i kombinasjon med if-else-setninger.
Eksempel: lager et tilfeldig tall mellom 10 og 20 (med tilfeldig-funksjonen!) og sjekker om det er et partall. Partall skal returne true, mens oddetall skal returnere false.
   < script >
     function sjekkPartall(min, max){
       var tilfeldig = tilfeldigTall(min, max);
       document.write(tilfeldig + '< br >');
       if (tilfeldig % 2 == 0){
         return true;
       }
       console.log('usant');
       return false;
     }
     if(sjekkPartall(10, 20) == 1){
       document.write('tallet er et partall');
     }
     else{
       document.write('tallet er et oddetall');
     }
   < /script >
