Hva er en if-setning?

En if-setning gir oss muligheten til å kjøre ulike deler av koden avhengig av forholdene som ligger till rette. Vi kan illustrere if-setninger med enkle flytdiagrammer:
Under vises flytdiagrammet for et program som genererer et tilfeldig tall og sjekker om det er et oddetall eller partall:

I dette diagrammet ligger det flere opplysninger:

  1. Generer først et tilfeldig tall.
  2. Beregner tall % 2
  3. Hvis tall % 2 er 0: skriv "tallet er et partall"
  4. Hvis tall % 2 er 1: skriv "tallet er et oddetall"

For å oversette dette til kode, må vi innom flere metoder. Heng med!

Å genererere tilfeldig tall

Metoden Math.random(); genererer et tilfeldig tall mellom 0 og 0,99999999999999999.... Dersom vi ønsker å ha et heltall, må vi

  1. Multiplisere det med den (øvre grensen + 1) for hvilket tall vi ønsker å ha (f.eks. multiplisere med 11 for å få tall fra 0 til 10)
  2. Runde tallet ned til nærmeste heltall for å slippe desimaltall. Dette gjøres med metoden floor (rund ned)

Altså:
   < script >
     var tilfeldig = Math.random();
     console.log(tilfeldig);
     tilfeldig = tilfeldig * 11;
     console.log(tilfeldig);
     tilfeldig = Math.floor(tilfeldig);
     console.log(tilfeldig);
   < /script >

Dette kan også skrives enklere:
   < script >
     var tilfeldigTall = Math.floor(Math.random()*11);
     console.log(tilfeldigTall);
   < /script >

Tips: Math.round() og Math.ceil()

Å sammenlikne tall

I forrige time gikk vi gjennom det vi kaller tilordningsoperatorer: de operatorene som tilordner en variabel en ny verdi. F.eks vil operatoren
   a = a + 3;
gi variabelen a verdien // gammel verdi + 3.
Sammenlikningsoperatorene skiller seg fra tilordningsoperatorene ved at de sjekker om en variabel har en gitt verdi:

Operator Forklaring Eksempel
== Lik verdi 10 == '10'; gir true
=== Lik verdi og lik datatype 10 === 10; gir true, 10 === '10'; gir false
!= Ikke lik verdi 10 != 5; gir true
!== Ikke lik verdi eller ikke lik datatype 10 !== '10'; gir true, 10 !== 5; gir true
> Større enn 10 > 9; gir true, 10 > 10; gir false
< Mindre enn 9 < 10; gir true; 10 < 10; gir false
>= Større eller lik 10 >= 10; gir true;
<= Mindre eller lik 10 <= 10; gir true;

Oppgaver

S 43: 3.1: Noter først hva dere forventer som resultat, deretter kan dere kjøre koden.

Å bruke if-setninger til å styre hvilken kode som kjøres

I en if-setning bruker vi sammenlikningsoperatorene som en forutsetning for kode som kjøres. If-setninger er strukturert som:
   < script >
     if(sammenlikningspåstand){
       kode som kjøres;
     }
   < /script >

For eksempel vil koden for å skrive til siden "Tallet er mindre enn 10" dersom et satt tall er mindre enn 10 skrives som:
   < script >
     var tall = 0;
     if(tall < 10){
       document.write('Tallet er mindre enn 10');
     }
   < /script >

Oppgave

Utvid koden til å sjekke om et tilfeldig tall mellom 0 og 100 er mindre enn 10

Else-if

Vi har også mulighet til å skrive if-setninger der ulike betingelser gir ulike konsekvenser:
   dersom betingelse1 == sann{
     kjør kode1;
   }
dersom betingelse1 != sann, mens betingelse 2 == sann{
     kjør kode2;
   }
   dersom ingen av disse er sanne{
     kjør kode3;
  }
// koden skrevet over kalles pseudokode og er en forklarende kode som ikke kun kan leses av mennesker, ikke av maskinen. Mer om dette senere!

Koden over vil skrives som:
   < script >
     if(betingelse1 == true){
       // kode som kjøres
     }
     else if(betingelse2 == true){
       // kode som kjøres
     }
     else{
       // kode som kjøres
     }

For eksempel kan vi sjekke om et tall mellom 0 og 100 er mindre enn 10, større enn 90 eller et sted i mellom:
   < script >
     var tall = Math.floor(Math.random()*101);
     console.log(tall);
     if(tall < 10){
       console.log('Tallet er mindre enn 10');
     }
     else if(tall > 90){
       console.log('Tallet er større enn 90');
     }
     else{
       console.log('Tallet er mellom 10 og 90');
    }
   < /script >

Hva hvis flere påstander skal være sanne?

Vi kan sette flere påstander som kriterie for if-setninger ved å bruke de logiske operatorene og (&&), eller (||) eller ikke (!). I eksemplene er a = 7, b = 3:

Operator Forklaring Eksempel
&& Og (begge påstander sanne) (a < 10 && b > 2); a mindre enn 10 og b større enn 2 - gir true
|| Eller (en av påstandene sanne) (a < 8 || b > 5); a mindre enn 8 eller b større enn 5 - gir true
! Ikke (påstanden er ikke sann) !(a < b); a er ikke større enn b - gir true

På den måten kan vi for eksempel gi betingelsen (tall mellom 0 og 50 er mindre eller lik 20 og delelig med 5):
   < script >
     var tall = Math.floor(Math.random()*51);
     console.log(tall);
     if(tall <= 20 && tall%5==0){
       console.log('Tallet er mindre enn 20 og delelig med 5');
     }
   < /script >
