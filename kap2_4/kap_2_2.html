De tre datatypene

String

Datatypen string brukes for tekstvariabler. All tekst skal legges i fnutter ('tekst') eller dobbeltfnutter ("tekst"):
   < script >
     var tekstVar = 'En setning med tekst!';    < script >
< /script >
En tekststreng er indexert fra 0 til (n-1) der n er antall tegn. Vi kan behandle tekstvariabler ved flere metoder, blant annet

Metode Forklaring Eksempel
[plass] Gir tegnet på en gitt plassering console.log(tekstVar[3]); returnerer s
indexOf() Finner plasseringen til første element av typen et spørres etter console.log(tekstVar.indexOf('n')); returnerer 1
toUpperCase() Gjør om til kun store bokstaver console.log(tekstVar.toUpperCase()); returnerer EN SETNING MED TEKST!
toLowerCase() Gjør om til kun små bokstaver console.log(tekstVar.toLowerCase()); returnerer en setning med tekst!
length Gir antall tegn i stringen console.log(tekstVar.length); returnerer 21
substring() Gir tegnene i et intervall console.log(tekstVar.substring(11, 14)); returnerer med

Boolean

Vi har to boolske variabler: sann (1) og usann (0). Disse legger blant annet grunnlag for if-setninger, der vi ber om at kode skal kjøres dersom en påstand er sann eller usann.
Boolske variabler lagres som
   < script >
     var boolskVar = true;
     var boolskVar = false;
     var boolskVar = 1;
     var boolskVar = 0;
   < /script >

Number

Number er alle tallvariabler: heltall, desimaltall, negative tall, positive tall. Merk at JavaScript ikke skiller mellom heltall og desimaltall!
Vi lagrer numbervariabler som
   < script >
     var tallVar = 1;
     var tallVar = 1.0;
     var tallVar = -157.42
   < /script >
