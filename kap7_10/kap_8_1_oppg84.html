Oppgaven

I oppgave 8.4 s. 127 skal dere
Opprette et < p >-element som bytter kantlinje hver gang det klikkes: fra gul til blå kantlinje rundt elementet.

Løsning

Her er det tilstrekkelig med internal css: i < style > legges følgende kode:
   body{
     font-family: "Century Gothic", Arial, sans-serif;
     font-size: 14px
   }
   #tekst {
     background-color: #f05a3f;
     padding: 20px;
   }
   #tekst:hover{
     cursor: pointer;
   }
   .gul{
     border: 4px solid yellow;
   }
   .blaa{
     border: 4px solid blue;
   }

Deretter legger vi til det aktuelle p-elementet, som jeg her gir iden tekst fordi jeg har flere p-elementer:
< p id = "tekst" > Klikk på dette tekstelementet og se hva som skjer! < /p >

Klikk på dette tekstelementet og se hva som skjer!

Deretter må vi i script legge til en lytter som sjekker om elementet klikkes. Første gang det skjer skal det få gul kantlinje, neste gang blå osv. Dette løser vi ved å bruke removeEventListener og addEventListener i hver hendelsesstyrte funksjon:
   < script >
     var tekstEl = document.querySelector("#tekst");
     tekstEl.addEventListener("click", gulKant);
     function gulKant(){
       tekstEl.setAttribute("class", "gul");
       tekstEl.removeEventListener("click", gulKant);
       tekstEl.addEventListener("click", blaaKant);
     }
     function blaaKant(){
       tekstEl.setAttribute("class", "blaa");
       tekstEl.removeEventListener("click", blaaKant);
       tekstEl.addEventListener("click", gulKant);
     }
   < /script >
