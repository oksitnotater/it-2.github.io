Oppgave

Denne fila viser kun hvordan oppgaven skal løses. Selve scriptet er laget i fila kap_7_eksempel_script.html
1. Opprett et nytt, tomt html-dokument.
2. Legg (som internal style sheet er ok) til klassene overskrifter og tom samt kantlinjer for tabeller:
   .overskrifter{
     font-family: Arial, sans-serif;
     font-size: 400%;
     color: green;
   }
   .tom{
     background-color: grey;
     width: 70%;
   }
   table, td{
     border: 1px solid black;
     border-collapse: collapse;
   }
3. Legg til følgende innhold ved hjelp av script:
   a) En overskrift som tilhører klassen overskrifter
   b) En nummerert liste med fire elementer
   c) Et img-element uten src
   d) Et tomt div-element med klassen "tom"
4. Gi lista rød skriftfarge og endre listetypen til å være lower-roman
5. Gi img-elementet src="http://wpstatic.idium.no/www.dooa.no/2017/08/Basse-Bj%C3%B8rndal-256x300.jpg" (eller annet valgfritt bilde) og bredde 400px
6. Fyll det tomme div-elementet med en 4*4 tabell som inneholder bokstaven A der tabellen fyller hele bredden til div-elementet

Løsning

Husk at scriptet i sin helhet ligger i egen fil

3. Legg til følgende innhold ved hjelp av script:
   // henter først inn body:
   var bodyEl = document.querySelector("body");

a) En overskrift som tilhører klassen overskrifter
   var overskriftEl = document.createElement("h2");
   overskriftEl.innerHTML = 'En litt mer sammensatt oppgave';
   overskriftEl.setAttribute("class", "overskrifter");
   bodyEl.appendChild(overskriftEl);

b) En nummerert liste med fire elementer
   var olListeEl = document.createElement("ol");
   for(var i = 0; i< 4; i++){
     var listeEl = document.createElement("li");
     listeEl.innerHTML = 'dette er listepunkt nr ' + (i+1);
     olListeEl.appendChild(listeEl);
   }
   bodyEl.appendChild(olListeEl);

c) Et img-element uten src
   var bildeEl = document.createElement("img");
   bildeEl.setAttribute("alt", "det finnes ikke noe bilde");
   bodyEl.appendChild(bildeEl);

d) Et tomt div-element med klassen "tom"
   var tomEl = document.createElement("div");
   tomEl.setAttribute("class", "tom");
   bodyEl.appendChild(tomEl);

4. Gi lista rød skriftfarge og endre listetypen til å være lower-roman
   olListeEl.style.color="red";
   olListeEl.style.listStyleType="lower-roman";

5. Gi img-elementet src="http://wpstatic.idium.no/www.dooa.no/2017/08/Basse-Bj%C3%B8rndal-256x300.jpg" (eller annet valgfritt bilde) og bredde 600px
   bildeEl.setAttribute("src", "http://wpstatic.idium.no/www.dooa.no/2017/08/Basse-Bj%C3%B8rndal-256x300.jpg");
   bildeEl.setAttribute("width", "400px");

6. Fyll det tomme div-elementet med en 4*4 tabell som inneholder bokstaven A der tabellen fyller hele bredden til div-elementet
   var divEl = document.querySelector(".tom");
   var tabellEl = document.createElement("table");
   for (i = 0; i < 4; i++){
     radEl = document.createElement("tr");
     for (j = 0; j < 4; j++){
      var kolonneEl = document.createElement("td");
       kolonneEl.innerHTML = "A";
       radEl.appendChild(kolonneEl);
     }
     tabellEl.appendChild(radEl);
   }
   tabellEl.style.width="100%";
   divEl.appendChild(tabellEl);
