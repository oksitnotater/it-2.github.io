Oppgave 10.4

Lag din egen versjon av "kalkulatoren" som ble beskrevet i teksten på forrige side
Kalkulatoren består av to tallfelt der brukeren skriver inn to tall som skal summeres samt en tom div der resultatet skrives ut:

Tall 1: Tall 2:

   Tall 1: < input type="number" id="tall1" >
   Tall 2: < input type="number" id="tall2" >
   < button id="summer" > Summer tallene < /button >
   < div id="resultat" >< /div >

I scriptet hentes verdiene til tallene ut, legges sammen, og resultatet skrives ut. Dette skjer når knappen klikkes:

   var summerEl = document.querySelector("#summer");
   var resultatEl = document.querySelector("#resultat");
   summerEl.addEventListener("click", summerFunksjon);
   function summerFunksjon(){
     var tall1El = document.querySelector("#tall1");
     var tall2El = document.querySelector("#tall2");
     var sum = Number(tall1El.value) + Number(tall2El.value);
     resultatEl.innerHTML = "< p > Summen av " + tall1.value + " og " + tall2.value + " er " + sum + "< /p >";
   }
