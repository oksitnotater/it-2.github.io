Enten fortsettelse bubblesort, grundig gjennomgang av bytter
Eller jobbe med en av sorteringsalgoritmene quick sort, selection sort eller merge sort. Dere kan i så fall ta utgangspunkt i lenken til ulike sorteringsalgoritmer på itslearning.

Bubble sort: hva skjer?

Vi har arrayen a = [1, -4, 5, 7, 2, 3]

    Første runde:
  1. Sammenlikner 1 og -4:
       if(1 > -4){
         // bytt plass
       }
    Nå er a = [-4, 1, 5, 7, 2, 3]
  2. Sammenlikner 1 og 5
       if(1 > 5){
         // bytt plass (skjer ikke)
       }
    Nå er a = [-4, 1, 5, 7, 2, 3]
  3. Sammenlikner 5 og 7
       if(5 > 7){
         // bytt plass (skjer ikke)
       }
    Nå er a = -4, 1, 5, 7, 2, 3]
  4. Sammenlikner 7 og 2
       if(7 > 2){
         // bytt plass
       }
    Nå er a = [-4, 1, 5, 2, 7, 3]
  5. Sammenlikner 7 og 3
       if(7 > 3){
         // bytt plass
       }
    Nå er a = [-4, 1, 5, 2, 3, 7]

Etter første runde vet vi helt sikkert at vi har flyttet arrayens største verdi bakerst.

    Andre runde
  1. Sammenlikner -4 og 1
       if(-4 > 1){
         // bytt plass (skjer ikke)
       }
    Nå er a = [-4, 1, 5, 2, 3, 7]
  2. Sammenlikner 1 og 5
       if(1 > 5){
         // bytt plass (skjer ikke)
       }
    Nå er a = [-4, 1, 5, 2, 3, 7]
  3. Sammenlikner 5 og 2
       if(5 > 2){
         // bytt plass
       }
    Nå er a = [-4, 1, 2, 5, 3, 7]
  4. Sammenlikner 5 og 3
       if(5 > 3){
         // bytt plass
       }
    Nå er a = [-4, 1, 2, 3, 5, 7]

Nå trenger vi ikke sammenlikne 5 og 7 fordi vi visste at den største verdien står bakerst.
Nå vet vi at de to største verdiene er de bakerste.

Slik forstetter sorteringen ved at man gjør en sammenlikning mindre hver gang. Siste runde sammenliknes kun de to første verdiene i arrayen, og da vet man at hele er sortert. Skrevet som kode trenger vi da en løkke som starter hver sorteringsrunde og en annen forløkke som sørger for at den neste runden alltid har en sammenlikning mindre enn den forrige.

Kode for bubble sort

Koden for bubble sort vises under. Her skriver jeg ut de delvis sorterte arrayene til siden underveis slik at dere får se alt som skjer (i stedet for å logge til konsollen):

   function sortereFunksjon (a){
     document.write("< b >Den usorterte arrayen er " + a +"< /b >< br >");
     for (var i= 0; i < a.length; i++){ // løkker gjennom fra første til siste element
       document.write("< b> Starter runde< /b >< br >");
       for (var j=0; j < a.length-i; j++){ // løkker gjennom fra første til siste-i element
         if (a[j] > a[j+1]){ // sjekker om tallet er større enn tallet etter, i så fall bytter de plass
           var temp = a[j]; // må lagre det største tallet i en midlertidig variabel når jeg bytter plass
           a[j] = a[j+1]; // setter det minste av tallene på plass j
           a[j+1] = temp; // setter det største av tallene på plass j+1
         }
       }
       document.write("< b >Neste runde< /b >< br >")
     }
     document.write("< b >Den sorterte arrayen er " + a+"< /b >< br >");
   }
   a = [1, -4, 5, 7, 2, 3];
   sortereFunksjon(a);
