Arrays

Så langt har vi sortert all data i enkeltvariabler, noe som er greit når vi ikke har mye data og når vi ikke skal gjenta funksjoner for flere variabler.
Se for dere følgende situasjoner:

  1. Dere skal finne gjennomsnittet av 100 tall
  2. Dere skal lage en quiz med 20 spørsmål der alle har radioknapper og like mange svaralternativer
  3. Dere skal sortere et sett med 50 tall i stigende rekkefølge
  4. Dere skal sortere klasselisten etter første navn i fornavnet

I alle situasjonene over kan man bruke mange enkeltvariabler, men dette blir mye å kode, og koden vil inneholde mange gjentakelser. Da er det enklere å bruke arrays, som er en samling av enkeltverdier.

Å deklarere arrays

Vi definerer en array på samme måte som en variabel:
   var tallArray =
Men i stedet for å gi den en enkeltverdi samler vi alt som skal lagres i arrayen mellom [] og skiller verdiene med komma:
   var tallArray = [0, 1, 2, 3, 4, 5];
Variablen over er array med lengde 6. Som for tekststrenger har første verdi index 0:
   tallArray[0] = 0;

Oppgave

Lag to arrayer: en array navn som inneholder navn på familiemedlemmene dine og en array alder som inneholder alderen deres. Skriv ut lengden av hver array (navn.length og alder.length) og den første og siste verdien i arrayen.

Array inni array

Vi kan også slå sammen navn- og alder-arrayene til én familiearray med to dimensjoner:
   var familie = [
     ['Asta', 'Christian'],
     [27, 29]
   ];

For å henge ut enkeltverdier vil nå
familie[0] returnere ['Asta', 'Christian']
familie[1] returnere [27, 29]
familie[0][0] returnere 'Asta'
familie[0][1] returnere 'Christian'
familie[1][0] returnere 27
familie[1][1] returnere 29

Oppgave

Lag en array som inneholder navn og alder på familiemedlemmene dine. Skriv ut lengden av arrayen og ett navn og én alder.

Å redigere arrays

Selv om vi deklarerer hele arrayen i ett, kan vi gå tilbake å redigere enkeltverier i arrayen, legge til enkeltverdier i arrayen eller fjerne enkeltverdier i arrayen.

Vi endrer verdier i en array ved:
   var tallArray = [0, 1, 2, 3, 4, 5];
   console.log(tallArray);
   tallArray[2] = 42;
   console.log(tallArray);

Vi legger til verdier i en array ved metodene arrayNavn.push(verdi) (legger til på slutten av arrayen) og arrayNavn.unshift(verdi) (legger til på starten av arrayen):
   var tallArray = [0, 1, 2, 3, 4, 5];
   console.log(tallArray);
   tallArray.push(42);
   console.log(tallArray);
   tallArray.unshift(42);
   console.log(tallArray);

Vi fjerner verdier i en array ved metodene arrayNavn.pop() (fjerner på slutten av arrayen) og arrayNavn.shift() (fjerner på starten av arrayen):
   var tallArray = [0, 1, 2, 3, 4, 5];
   console.log(tallArray);
   tallArray.pop();
   console.log(tallArray);
   tallArray.shift();
   console.log(tallArray);

Vi finner enkeltverdier i en arrray ved metoden arrayNavn.indexOf(verdi) som gir oss plasseringen til den første forekomsten av en gitt verdi (senere skal vi klare å finne alle):
   var tallArray = [0, 1, 2, 3, 4, 5];
   console.log(tallArray);
   console.log(tallArray.indexOf(2));
   tallArray.unshift(2);
   console.log(tallArray);
   console.log(tallArray.indexOf(2));

Vi fjerner enkeltverdier midt i arrayen ved metoden arrayNavn.splice(index, antall verdier). Her kan indexen finnes i samme operasjon: arrayNavn.splice(arrayNavn.indexOf(verdi), antall verdier):
   var tallArray = [0, 1, 2, 3, 4, 5];
   console.log(tallArray);
   tallArray.splice(tallArray.indexOf(1), 1);
   console.log(tallArray);

Vi kan også legge til enkeltverdier midt i arrayen ved bruk av arrayNavn.splice(indexForForsteNye, antallSlettes, verdiSomLeggesTil, verdiSomLeggesTil osv.)
   var tallArray = [0, 1, 2, 3, 4, 5];
   console.log(tallArray);
   tallArray.splice(3, 0, 42, 42, 42);
   console.log(tallArray);
   tallArray.splice(3, 2, 11, 12, 'tall? tull?');
   console.log(tallArray);
