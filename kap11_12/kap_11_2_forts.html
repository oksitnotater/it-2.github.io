Oppgaver fra forrige time

1. Lag en funksjon som finner det minste tallet i en array.
   var tallArray = [4, 3, 11, 7, 9, -3, 4];
   var minst = tallArray[0];
   var plass = 0;
   for(var i = 0; i < tallArray.length; i++){
     if(tallArray[i] < minst){
       minst = tallArray[i];
       plass = i;
     }
   }
   console.log('Det minste tallet er '+minst+' og står på plass '+plass);

2. Lag en funksjon som finner summen av tallene i en array.
   var sum = 0;
   for(var i = 0; i < tallArray.length; i++){
     sum += tallArray[i]
   }
   console.log('Summen av tallene i arrayen er '+sum);

3. Lag en funksjon som finner gjennomsnittet av tallene i en array.
   var snitt = sum / tallArray.length;
   console.log('Gjennomsnittet av tallene i arrayen er '+snitt);

Løkke gjennom todimensjonale arrays

Todimensjonale arrays er arrays med arrays inne i:
   var A = [
     [1, 2, 3],
     [4, 5, 6]
   ]

For å løkke gjennom disse må vi bruke to for-løkker:
   for(var i = 0; i < A.length; i++){
     for (var j = 0; j < A[i].length; j++){
       console.log(A[i][j]);
     }
   }

Metoden querySelectorAll

Vi husker fra tidligere at metoden querySelector finner første forekomst av et element (querySelector("p")), av et element med en gitt klasse (querySelector(".klassenavn")) eller av et element med en gitt id (querySelector("#idnavn")). Det ble også nevnt at det finnes en metode querySelectorAll som tar inn alle elementer av en gitt type/klasse. Denne har vi spart fordi disse variablene lagres som nettopp en array. For å bearbeide dem må jeg da hente inn alle ved hjelp av en løkke.

Som eksempel kan jeg hente inn alle td-elementer i dette dokumentet og gi dem rosa bakgrunn:
   var tdEl = document.querySelectorAll("td");
   for(var i = 0; i < tdEl.length; i++){
     tdEl[i].style.backgroundColor = "pink";
   }

Hvilke idretter liker du?

Kunstløp

HTML for dette:
   < input type="checkbox" class="idrett" value="fotball">Fotball
   < input type="checkbox" class="idrett" value="kunstløp">Kunstløp
   < input type="checkbox" class="idrett" value="langrenn">Langrenn
   < input type="checkbox" class="idrett" value="turn">Turn
   < input type="checkbox" class="idrett" value="skihopping">Skihopping
   < input type="checkbox" class="idrett" value="vektløfting">Vektløfting
   < button id="send">Send svar < /button >
   < p id="skrivUt">< /p >

Scriptet for dette:
   var idrettEl = document.querySelectorAll(".idrett");
   var skrivUtEl = document.querySelector("#skrivUt");
   var sendEl = document.querySelector("#send");
   sendEl.addEventListener("click", sjekkSkjema);
   function sjekkSkjema(){
     skrivUtEl.innerHTML += 'Du liker å drive med < ul >';
     for(i = 0; i < idrettEl.length; i++){
       if(idrettEl[i].checked){
         skrivUtEl.innerHTML += '< li >' + idrettEl[i].value + '< /li >';
       }
     }
     skrivUtEl.innerHTML += '< /ul >';
   }
